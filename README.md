### 当前版本说明
很久以前，人们所生活的世界还没有任何高科技元素，只有最原始的"气"。人类中的智者开始思考如何利用"气"来完成一些事情。这就是他们的成果――修炼，让自己变得超乎凡人。

你作为学习修炼的学徒，需要在一批顶尖的智者所设立的地下城中闯关，到达关底即为正式入修炼之道。为了你的前途，向越来越深的地下冲吧！

按照说明操作即可。注意此版本的界面中“商城”和“个人信息”还没有开发，敬请期待。
